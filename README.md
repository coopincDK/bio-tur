# 🎬 Bio Tur Bestillingsseddel

En simpel, mobilvenlig web-app til at samle bestillinger til bio-turen.

## ✨ Features

- 👥 **Deltager liste** — forudfyldt med alle navne, tilføj/slet frit
- 🥤 **Drik** — Cola, Sport, Appelsinjuice, Slushice (Rød/Blå/Blandet), Vand, Andet
- 🍿 **Snack** — Popcorn, Baconchips, Blandet, Andet
- 💬 **Kommentarer** til både drik og snack
- 📊 **Oversigt** — se alle bestillinger + totaler + hvem mangler
- 🖨️ **Print-venlig** oversigt
- 💾 **LocalStorage** — gemmer automatisk på enheden (ingen server nødvendig)

## 🚀 Brug

Åbn `index.html` i en browser — eller host på GitHub Pages.

### GitHub Pages

1. Push til GitHub
2. Gå til **Settings → Pages**
3. Vælg branch `main`, folder `/root`
4. Del linket med alle deltagere

> **Note:** Hver enhed gemmer sine egne data i localStorage.
> Appen er designet til at én person (f.eks. Martin) samler alle bestillinger.

## 📱 Mobil

Fungerer perfekt på mobil — responsivt design med touch-venlige knapper.
